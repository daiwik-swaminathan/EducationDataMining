# EducationDataMining

## Setup

1. Clone the repository

```bash
git clone https://github.com/yourusername/EducationDataMining.git
```

2. Install the dependencies

```bash
pip install -r requirements.txt
```

3. Run the code

```bash
python predictions.py
```

The data is stored in the `Data` folder.
